# Checklist Curso Bienestar - Vértice

Tablero de seguimiento del curso Bienestar (Psicología Positiva) producido por Griky para Vértice.

## Cómo usar

### Para el equipo (consulta)
- Abre el link del sitio publicado
- Verás el estado oficial actualizado por Jorge
- Es solo lectura

### Para el editor (Jorge)
1. Abre la página agregando `?edit=jorge` al final de la URL  
   Ejemplo: `https://tuusuario.github.io/checklist-bienestar/?edit=jorge`
2. Haz click en **✏️ Modo edición**
3. Marca los estados (vacío → amarillo → verde) en cada celda
4. Tus cambios se guardan automáticamente en tu navegador
5. Cuando termines, click en **📤 Descargar para publicar**
6. Sube el archivo `estado.json` descargado al repo (reemplaza el existente)
7. El equipo verá los cambios en su próxima recarga

## Archivos del repo

- `index.html` — la página del checklist
- `estado.json` — estado oficial publicado (este es el archivo que actualizas)
- `README.md` — este documento

## Setup en GitHub Pages

1. Crea un repo nuevo (ej. `checklist-bienestar`)
2. Sube los 3 archivos
3. Ve a Settings → Pages
4. En "Source" selecciona `main` branch y carpeta `/ (root)`
5. Espera 1-2 minutos a que se publique
6. Tu link será `https://tuusuario.github.io/checklist-bienestar/`
